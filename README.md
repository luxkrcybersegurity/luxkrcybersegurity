<h2 align="center">Sobre LUXKR+</h2>

<picture>
 <img align="right" src="https://raw.githubusercontent.com/matfantinel/matfantinel/master/code.gif" width="300">
</picture>

LUXKR+ Security es un hub técnico enfocado en seguridad para el ecosistema **Stellar + Soroban** en LATAM.  
Combina auditorías full-stack, formación técnica y herramientas ofensivas/defensivas orientadas al desarrollo seguro.

Operamos en tres verticales estratégicas:
- Auditorías de Smart Contracts Soroban
- Evaluación de infraestructura Web2 (backend, APIs, bases de datos, redes)
- Formación continua para talento técnico mediante talleres y CTFs

<br/>

---

<h2 align="center">Qué hacemos</h2>

- Auditoría Web2/Web3 en múltiples capas
- Seguridad de Smart Contracts (Soroban / Rust)
- Modelado de amenazas e ingeniería de seguridad
- Tests de penetración + hardening de APIs y backend
- Consultoría en implementación segura y AppSec
- Entrenamiento técnico para equipos (hands-on, CTFs, workshops)
